# app-elbosque

Monorepo para múltiples proyectos y aplicaciones.

## Estructura

- `apps/` — Aplicaciones principales (frontend, backend, etc.)
- `packages/` — Librerías y utilidades compartidas
- `docs/` — Documentación general
- `scripts/` — Scripts de automatización

Cada app o paquete tiene su propio README y configuración.
