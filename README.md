# app-elbosque

Repositorio monorepo para la plataforma “app elbosque”. Este espacio integra múltiples proyectos y aplicaciones (frontend, backend, utilidades y documentación) orientados a la innovación, el aprendizaje y la gestión empresarial.

## Estructura del repositorio

- `apps/` — Aplicaciones principales (frontend, backend, etc.)
- `packages/` — Librerías y utilidades compartidas
- `docs/` — Documentación general
- `scripts/` — Scripts de automatización

Cada app o paquete tiene su propio README y configuración.

## Tecnologías principales

- React, Vite, Node.js, Express, etc.
- Git & GitHub para control de versiones
- Vercel para despliegue continuo

## ¿Cómo empezar?

1. Clona el repositorio:  
   git clone https://github.com/tuusuario/app-elbosque.git
2. Instala dependencias en cada app:  
   cd apps/AppPizzaCasera  
   npm install
3. Ejecuta la app:  
   npm run dev
